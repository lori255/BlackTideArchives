# 黑潮档案局_文档包_v0.2

本压缩包只包含文档、说明、索引和前端接入参考，不包含运行时美术素材图片。

## 目录说明

- `01_需求文档/`：游戏需求文档 PRD，包含 Word / Markdown / PDF。
- `02_素材包说明文档_v0.6/`：Web 开发素材包 v0.6 的说明、补齐报告和缺口报告。
- `03_前端接入参考/`：CSS tokens、组件样式、资源加载示例、Vue 组件示例。
- `04_预览与索引/`：素材 manifest 与 asset index，用于查看素材命名和路径。

## 注意

当前已有完整 PRD。独立的 GDD 游戏设计文档、UI/UX 设计规范、美术风格规范尚未单独成稿；现阶段相关内容分散在 PRD 与素材包说明文件中。
